# permu

```
└─$ python3 permu.py -h                                                                      130 ⨯
usage: gen.py [-h] [-l L] [-e E] [-w W] [-v]

Generate simple permutations from words.

required arguments:
  -w W        path to input wordlist, each word separated by newline

optional arguments:
  -h, --help  show this help message and exit
  -l L        max length of permutations
  -e E        exact permutation length to generate. if supplied, -l option is ignored.
  -v          verbose
```
